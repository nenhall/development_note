# 走过幽暗的岁月

```
    岁月远去，我无处安放蒙尘的心。这尘世烟花，太过绚丽，我本是尘埃，不愿刹刹那凋零。也许，岁月的尽头，阳光明媚，温暖永远。但世间，人情太冷，何必苦苦寻觅一心一意的真呢？我故作潇洒，可最后却挣脱不了岁月的束缚。我开始相信，我是自由的囚徒，只是不懂岁月给我的那些幽暗日夜罢了！
    时常心情郁结，总是走不出岁月的故事设定。那些脱离不了世俗的情节，总是在七情六欲中，翻来翻去。于是，我想刻意绕过人生的某些路程，走到自己想要的地老天荒。看着别人的幸福与成长，偶尔也禁不住人生失败的忧伤。这些年，一味的逃避，想踏破世人的寻常人生，活出不一样的真我。在经过一夜夜的沉思后，我才恍然明白，我所走过的岁月，不过是别人眼中的徘徊不前。

    连所钟爱的文字也可以放下的我，也许并没有更多的选择余地。就算我放下所有，向内心一往无前，我也不会遇见那个自己。有时，身处人海，迷失的感觉油然而生。我不喜欢人生的重叠与覆盖，更不喜欢为了那惊涛拍岸，将自己尽付云烟。尽管我是一滴咸涩的海水，但我也生机盎然。那无尽的深邃天空，必将是我抛头洒血的疆场。我要策马冲锋，用血肉之躯去填埋生命的疑惑。那样的我，不再茫然，心中只会徒增一份岁月清欢。

    遥望岁月，清明若水。既看不清，也道不明，只是一去不复返的向东而流。我携手尘世悲欢，用虚假的笑容，演绎真情世界。所有的鲜花与掌声，都如岁月的灰屑，纷飞发间。我笑而不语，用心灵的热忱感化内心的悲凉。此刻，我决定迈步风尘，一解此恨。 可我的世界，草长莺飞的惬意，没有为我带来丝毫的恬淡。我只能在每个静谧的夜里，与钟意的岁月擦肩而过。

    置身云水，洗尽铅华。我本以为禅音绵绵，情意悱恻。如今看来，青灯长眠，只不过是为古刹守一夜岁月罢了！那把穿过岁月的古琴，琴弦深处的哀痛，有谁能懂？郁郁檀香，随人间烟火，一同熏染我的心灵。我端坐岁月的花蕊深处，准备一场惊世骇俗的多情花开。冰凉的露珠涤荡我心，湿润了我的眼眸。我泪眼婆娑，静待春风化雨的那一刻温暖。那一次，我没有迎来人生的花开，只是浪费了一次不必要的等待。从此，我虚掩心扉，不再推窗聆听岁月呜咽弹唱。

    离开岁月，我是幽暗的黑夜。黑夜里，我忘了转瞬回眸，也忘了零落成泥。渐渐地，笔墨间的我，恰逢岁月的黄昏，坐等夕阳梦里那隽永的黄花瘦。夕阳的余晖，落在我颓废的面孔上，我顿感岁月的无情，也是如此的犹豫不决。从此，我再也没有回到那个黄昏，因为，我生命中的那个夕阳，再难续写我的狂妄岁月。我也知道，那个夕阳它必将如浴火凤凰，重获新生。我也要冲破今后生命力的每一个黑夜，站在黎明的晨曦下，等待清晨的夕阳。它如出水的芙蓉，不惹世间丝毫风尘。我相信，它是岁月的馈赠，也是黑夜凋零后的芳香果实。

    心的流浪，是岁月的宿命。哪怕归途茫茫，我也不应该读懂沧桑？如果人的成长，一定要用岁月去抵挡，那就请让我，用热泪盈眶，用方向代替彷徨。如今四处战火燃烧，我这个生活的逃兵，是否还有机会和岁月并肩作战？我拖着沉重的猎枪，在黑夜中前行。我坐在泛黄的枝叶上，用秋天的萧瑟心情擦亮猎枪，等待给岁月致命一击。我的背叛，必须用鲜血偿还。我不会吝惜这一身皮肉，只愿倒下去的自己，能用最诚挚的心跳，换回大地冬季后的生命复苏。就算我不再醒来，我也不会带走这个世界的一丝美好，因为我知道，我的梦，早已被岁月镂空。

    隔着岁月，我为自己编织生命的篱笆，阻挡风霜的浸染。四季不再为我更替，我只是顺着花草的年轮，一遍遍的循环着自己的快乐。如此的人生，短暂中，少了苦酒的咸涩，却多了几分水的醇香。我不懂品味，但我能饮下岁月最深的痛，独自感动。这一切，也许少了风浪的磨砺，但却多了随心所欲的生活舒畅。人活一世，不问大漠孤烟能有多直，只想看看滚滚长江能于岁月的河流中淘尽多少浪沙？苦短人生，若能有一梦，也不必在三更静夜里，辗转难眠，难以入睡。人的心事，是人说不出口的坚强和守不住的秘密。那些肆意妄为的心绪，会出卖你多彩的笑容。因而，忧伤的时候，紧锁愁眉也是一种释然的快乐。不必介怀于心，明白生死十年，终究抵不过岁月的那妩媚一笑。即使抓住了又如何，没有你的一席之地，你也不过是一名岁月过客。那些如歌岁月，不是每个人的声音都适合吟唱。宁愿一个人天真烂漫的寂寞，也不要打破无人问津的孤独。

    心的孤独，是岁月的惩罚。如果没有向死而生得勇气，就不要翻阅生命的历程。密密的文字，是你人生所不能承受的繁重。不如合上生命的书本，用心仰望无限的岁月。这是人生的沉淀，没有尘嚣的打扰，只是一个人内心安静的独白。也许，你不会就这样认识自己。因为，你还没有告诉自己，那些没有走过的千山万水，只是一些岁月的幽暗，只是一些你所向往的岁月艳阳。

    不管如今你身在何处，我们都在行走。岁月的眼看着我们的摔倒与扑扑泥土的站起，也许接下来的日子，还要摔倒，但必须义无反顾的站起。那些你慵懒的借口，只会让你最终命丧岁月的刀下。如果不畏生死，任岁月如何锋利，你也能从容面对，全身而退。只要放下了所有的欲望，你也不再欠岁月任何的欢笑。你只需每日晨钟暮鼓，遨游世外，逍遥自在。没有人再与你有任何故事交集，你只能在花开花落中，欣赏岁月的云卷云舒。

    岁月是姹紫嫣红后的一夜炎凉，无法温暖。岁月是繁华落尽后的一抹忧伤，挥之不去。岁月，也许幽暗，不见明媚艳阳，但也请在每个落魄的夜里，安静的缝合创伤。因为，岁月带我们走过的路，不止幽暗，还有看不见的灿烂。
```
